# Bootstrap
网格最多分为 12 列

在栅格参数中，有一个集中类，对于一个部分占用的列数，可以用集中类型来描述，分别是 col-xs, col-sm, col-md, col-lg
分别对应
.col-xs-* 超小屏幕 手机 $(<768px)$
.col-sm-* 小屏幕 平板 $(\geq 768px)$
.col-md-* 中等屏幕 桌面显示器 $(\geq 992px)$
.col-lg-* 大屏幕 大桌面显示器 $(\geq 1200px)$
