# `#` 和 `%` 操作符

`%` 操作符可以轻松将名称部分从 `名称.扩展名` 这种格式中提取出来。`#` 可以提取扩展名。

---

1. 提取文件名

    ```sh
    echo ${file_name%.*}
    ```

    `${VAR%.*}` 表示删除位于 `%` 后侧的通配符所匹配的字符串。通配符从右向左进行匹配。

    `%` 属于非贪婪操作，它从右到左找出匹配通配符的最短结果。`%%` 与 `%` 类似，但是其行为模式是贪婪的。

2. 提取扩展名

    ```sh
    echo ${file_name#*.}
    ```

    `${VAR#*.}` 表示删除位于 `#` 后侧的通配符所匹配的字符串。通配符从左往右进行匹配。

    `#` 属于非贪婪操作，而 `##` 属于贪婪操作。
