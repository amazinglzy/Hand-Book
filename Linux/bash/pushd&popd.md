# pushd & popd

`pushd` 和 `popd` 可以用于在多个目录之间进行切换而无需复制并粘贴目录路径。

---

1. 压入并切换路径

   ```sh
   pushd /var/www
   ```

2. 用下面命令查看栈的内容

   ```sh
   dirs
   ```

3. 切换路径

   ```sh
   pushd +3
   ```

   该命令会将栈上方的几个路径翻转。

4. 删除路径

   ```sh
   popd
   ```

5. 删除前几个路径

   ```sh
   popd +3
   ```
